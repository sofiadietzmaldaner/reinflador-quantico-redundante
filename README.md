# Reinflador Quântico Redundante

Trabalho desenvolvido para a disciplina de Algoritmos e Estruturas de Dados da PUCRS.

O projeto apresenta três abordagens diferentes para resolver o problema do **Reinflador Quântico Redundante**, cujo objetivo é identificar a letra inicial responsável pela expansão e calcular o tamanho final do arquivo reinflado sem construir explicitamente a string resultante.

Todas as soluções foram implementadas em **Python** e exploram diferentes estratégias algorítmicas:

- **Primeira solução:** recursão com memoização;
- **Segunda solução:** versão otimizada da abordagem recursiva;
- **Terceira solução:** programação dinâmica iterativa (*bottom-up*).

O repositório inclui:
- códigos-fonte das três soluções;
- testes realizados;
- análise de complexidade;
- comparação de desempenho entre as abordagens.
